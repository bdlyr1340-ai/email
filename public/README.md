# CD Store — UI Prototype

نسخة تصميم وتجربة استخدام جديدة بالكامل.

## المحتويات
- `index.html`: واجهة المتجر للزبون.
- `admin.html`: لوحة إدارة ديناميكية.
- `assets/styles.css`: التصميم المتجاوب.
- `assets/store.js`: عرض المنتجات، اللغة، البحث، خيارات المنتج.
- `assets/admin.js`: إضافة/تعديل/حذف المنتجات وحفظها محلياً.

## التشغيل
افتح `index.html` في المتصفح.
لوحة الإدارة عبر `admin.html`.

## ملاحظة
هذه نسخة واجهات تعمل بـ LocalStorage حتى نثبت الشكل وتجربة الاستخدام قبل ربط:
PostgreSQL، تسجيل الدخول، Binance Pay، SuperQi، OpenAI، Telegram، والمخزون الحقيقي.
